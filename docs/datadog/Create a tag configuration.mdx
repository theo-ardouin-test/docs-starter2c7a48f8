<h1 className={"openapi__heading"}>Create a tag configuration</h1>

<MethodEndpoint method={"post"} path={"/api/v2/metrics/{metric_name}/tags"}></MethodEndpoint>


Create and define a list of queryable tag keys for an existing count/gauge/rate/distribution metric.Optionally, include percentile aggregations on any distribution metric or configure custom aggregationson any count, rate, or gauge metric.Can only be used with application keys of users with the `Manage Tags for Metrics` permission.
