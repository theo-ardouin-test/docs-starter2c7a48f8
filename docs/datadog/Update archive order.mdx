<h1 className={"openapi__heading"}>Update archive order</h1>

<MethodEndpoint method={"put"} path={"/api/v2/logs/config/archive-order"}></MethodEndpoint>


Update the order of your archives. Since logs are processed sequentially, reordering an archive may changethe structure and content of the data processed by other archives.**Note**: Using the `PUT` method updates your archive's order by replacing the current orderwith the new one.
